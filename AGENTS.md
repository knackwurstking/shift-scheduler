# Shift Scheduler

## Stack
- **Go 1.25** with Echo web framework
- **Templ** for HTML templating (generates Go code)
- **Go WASM** frontend (pointer event handlers exported to JS)

## Build Commands
```bash
make init    # Run templ generate, copy wasm_exec.js, go mod tidy
make build   # Compile WASM + server binary
make run     # init + build + start server on :3000
```

## Architecture
- `cmd/shift-scheduler/` - Echo web server (serves `assets/public/` as static)
- `cmd/wasm/` - WASM binary (exports `onPointerDown`, `onPointerMove`, `onPointerUp`, etc.)
- `templates/` - Templ components with subdirs `components/` and `pages/`
- `internal/handlers/` - HTTP handlers
- `internal/localization/` - i18n (en/de via Accept-Language header)
- `assets/public/` - Embedded static assets (WASM, CSS)

## Generated Files (gitignored)
- `*_templ.go` - Generated by `templ generate`
- `assets/public/main.wasm` - Compiled WASM
- `assets/public/wasm_exec.js` - Copied from GOROOT

## Environment Variables
- `ADDRESS` - Unused (server hardcoded to `:3000`)
- `VERBOSE=true` - Enable debug-level logging
