# Shift Scheduler

## Stack
- **Go** with Echo web framework
- **Templ** for HTML templating (generates Go code)
- **Go WASM** frontend (compiled to `assets/public/main.wasm`)
- **Go 1.25**

## Build Commands
```bash
make init    # Run templ generate, copy wasm_exec.js, go mod tidy
make build   # Compile WASM (GOOS=js GOARCH=wasm) + server binary
make run     # init + build + start server
```

## Architecture
- `cmd/shift-scheduler/` - Echo web server entrypoint
- `cmd/wasm/` - WASM frontend (pointer event handlers)
- `templates/*.templ` - Templ components (generate `*_templ.go`)
- `internal/handlers/` - HTTP handlers
- `internal/localization/` - i18n (en/de via Accept-Language header)
- `assets/public/` - Embedded static assets (WASM, CSS)

## Generated Files (gitignored)
- `*_templ.go` - Generated by `templ generate`
- `main.wasm` - Compiled WASM output
- `wasm_exec.js` - Copied from GOROOT during `make init`

## Environment Variables
- `ADDRESS` - Server address (default `:3000`)
- `VERBOSE` - Enable debug logging (`true`/`false`)
